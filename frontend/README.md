# Perplexity Clone Frontend

This is the frontend repository for the Perplexity clone, built with Next.js 15 (App Router), Tailwind CSS, and TypeScript.

## Getting Started

First, make sure the backend is running on `http://localhost:3000`.

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3001](http://localhost:3001) with your browser to see the result.

See the `docs/` directory for the roadmap and technology stack.
